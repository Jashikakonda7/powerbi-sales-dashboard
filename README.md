# 📊 Sales Dashboard (Power BI)

## 🔍 Overview

Developed an interactive Sales Dashboard using Power BI to analyze business performance and generate actionable insights.

---

## 🛠 Tools & Technologies

* Power BI
* Data Cleaning
* Data Visualization
* DAX (Basic Measures)

---

## 📊 Features

✔ KPI Cards (Total Sales, Profit, Orders)
✔ Sales by Region and Category
✔ Monthly Sales Trend Analysis
✔ Top Products Performance
✔ Interactive Filters (Year, Category)

---

## 📈 Key Insights

* West region contributed highest sales
* Technology category generated maximum revenue
* Seasonal trends observed in monthly sales
* Discounts impacted overall profitability
